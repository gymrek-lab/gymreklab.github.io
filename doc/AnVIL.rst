AnVIL for dbGAP data analysis
======================================================

Last update: 2025/03/23

Getting access to the dbGAP 
----------------------------------------------
You need to do both to get access to dbGAP data from AnVIL

#. Email Melissa to: 1) get you eRA commons account for dbGAP login; 2) confirm that we have access to the data through dbGAP (usually by submit a project application) and to add you as an authorized user.

Setting up the AnVIL account
----------------------------

#. `Create an AnVIL account  <https://anvil.terra.bio>`_ using your UCSD email. Note, you'll need to link the AnVIL account to the dbGAP account and gmail is not allowed to link.
#. Link the Terra account to NIH account using this `instruction <https://support.terra.bio/hc/en-us/articles/19124069598235-Access-controlled-data-files-by-linking-your-NIH-account-in-Terra>`_. The link needs renew every 15 days. AnVIL just updated the data link regulation, it requires use the :code:`login` account linked to eRA, for details follow this `link <https://support.terra.bio/hc/en-us/articles/32634034451099-RAS-Integration-for-AnVIL-Data-Released-3-25-26>`_ for link dbGAP access to AnVIL.
#. Ask Melissa or Yang to create an AnVIL workspace for you to work on.
